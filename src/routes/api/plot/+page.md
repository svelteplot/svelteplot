---
title: Plot API reference
---

The Plot component is the container for plots. It collects the marks with their data and channels and computes the shared scales. The Plot component is split into two parts. This is the outer Plot which provides convenient defaults and automatically adds axes etc to the graphics. The downside is that it adds a bunch of imports that you may not be using. To help with this you can use the core/Plot component directly for a more low-level Plot wrapper.

### ScaleOptions

| Prop           | Type                                                                                                                             | Description                                                                                                                                  |
| -------------- | -------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| `type`         | ScaleType \| 'auto'                                                                                                              | Override the automatic scale type detection.                                                                                                 |
| `domain`       | RawValue[]                                                                                                                       | Set a custom domain for the scale instead of auto-computing the domain<br>from the mark data channels.                                       |
| `range`        | RawValue[]                                                                                                                       | Set a custom range for the scale.                                                                                                            |
| `reverse`      | boolean                                                                                                                          | Reverse the scale.                                                                                                                           |
| `label`        | string \| false                                                                                                                  | the axis label for this scale, or false to disable the automatic label                                                                       |
| `interval`     | string \| number                                                                                                                 | the interval for ordinal/temporal scales, e.g. 'day', 'month', or a numeric step                                                             |
| `clamp`        | boolean                                                                                                                          | clamp values outside the domain to the nearest domain boundary<br>(applicable to quantitative scales only)                                   |
| `nice`         | boolean                                                                                                                          | Extend the domain to nice round numbers (applicable to quantitative scales only)                                                             |
| `zero`         | boolean                                                                                                                          | Include zero in the scale domain (applicable to quantitative scales only)                                                                    |
| `round`        | boolean                                                                                                                          | round interpolated values to integers (applicable to quantitative scales only)                                                               |
| `percent`      | boolean                                                                                                                          | format values as percentages and append (%) to the axis label<br>(applicable to quantitative scales only)                                    |
| `transform?`   | (d: RawValue) =&gt; RawValue                                                                                                     | custom transformation function applied to values before scaling                                                                              |
| `padding`      | number                                                                                                                           | set the padding for band scales                                                                                                              |
| `align`        | number                                                                                                                           | set the align for band or point scales                                                                                                       |
| `paddingInner` | number                                                                                                                           | set the inner padding for band scales                                                                                                        |
| `paddingOuter` | number                                                                                                                           | set the outer padding for band scales                                                                                                        |
| `insetLeft`    | number                                                                                                                           | reduce the scale range by this many pixels on the left side                                                                                  |
| `insetRight`   | number                                                                                                                           | reduce the scale range by this many pixels on the right side                                                                                 |
| `insetTop`     | number                                                                                                                           | reduce the scale range by this many pixels on the top side                                                                                   |
| `insetBottom`  | number                                                                                                                           | reduce the scale range by this many pixels on the bottom side                                                                                |
| `ticks`        | (number \| Date)[]                                                                                                               | explicit tick values to use instead of automatic tick generation                                                                             |
| `tickSpacing`  | number                                                                                                                           | desired pixel distance between ticks for automatic tick generation                                                                           |
| `base`         | number                                                                                                                           | the logarithm base for log scales (default 10)                                                                                               |
| `sort`         | ChannelAccessor \| ((a: RawValue, b: RawValue) =&gt; number) \| {'{'} channel: string; order: 'ascending' \| 'descending'; {'}'} | controls domain ordering for band and point scales; can be a channel<br>accessor, a comparator function, or an object with channel and order |
| `constant`     | number                                                                                                                           | the constant parameter for symlog scales, controlling the transition<br>between linear and logarithmic behavior near zero (default 1)        |
